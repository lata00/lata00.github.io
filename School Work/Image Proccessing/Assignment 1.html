Introduction

For this assignment, I would like to write an algorithm that will take a section from a larger image as an input and determine the coordinates of that portion within the larger image itself. For example, given this image:

Figure 1: "Big" Image

I would like to write an algorithm that will find the coordinates of this section of the image:

Figure 2: Image to be located in large image

In order to do this, I can make use of the normalized cross-correlation function, which is defined as:

Figure 3: Normalized Cross-Correlation Definition [1]

This function will pass the sample image over the original and calculate the average value of how well the correlate to one another at that point, as shown in Figure 4. Since this is the normalized correlation, the point at which the two arrays are equal should yield a value of 1 from the correlation calculation.

Algorithm

Fortunately, MATLAB includes a function for performing 2-dimensional normalized correlations between 2 arrays: normxcorr2. Using this, the algorithm for finding the location of the sample image inside the larger image is very straightforward. First, we must convert the color images to grayscale, since the correlation can only be performed between two arrays, and the color images are comprised of 3 separate arrays. This can be done with the rgb2gray function:

f = rgb2gray(big_im);

w = rgb2gray(object_im);

 

Now, we can calculate the correlation between these two images:

 

g = normxcorr2(w,f);

 

The result of this operation is another 2-dimensional array that has dimensions corresponding roughly to the sum of the dimensions of the two arrays being correlated. A sample correlation is shown here:

Figure 5: Sample Correlation

Looking closely, you can see a white dot near the top-left corner. The next step will be to find the coordinates of this spot. This can be done with the following code:

 

[r,c] = find(g==max(g(:)));

 

However, since the dimensions of the correlation dont match the original image, this will not give us the correct coordinates relative to the image we are interested in. To get back to our original coordinate frame of reference, we must subtract the size of the sample image out of the coordinates we just determined:

r = r - Ro;

c = c - Co;

 

This should now give us the location in the original image for which the two images are most correlated. Running the main function with this algorithm as described yields the following results:
